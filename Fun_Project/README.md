# Number Guessing Game

## Description
A simple game where the user tries to guess a randomly generated number between 1 and 100.

## Dataset
- No dataset required

## Steps Performed
1. Generate a random number
2. Take user input
3. Compare the guess
4. Repeat until correct

## Results
- User successfully guesses the number

## Tools Used
- Python
- random module

## Conclusion
This project demonstrates basic Python concepts like loops and conditions.
