# House Price Prediction

## Description
This project predicts house prices based on features like area and number of bedrooms using a simple linear regression model.

## Dataset
- Source: Manually created dataset
- Contains area, number of bedrooms, and price

## Steps Performed
1. Data Creation
2. Data Preparation using Pandas
3. Splitting data into training and testing sets
4. Model Building using Linear Regression
5. Prediction

## Results
- Model successfully predicts house prices
- Demonstrates relationship between features and price

## Tools Used
- Python
- Pandas
- Scikit-learn

## Conclusion
This project shows how machine learning can be used to predict house prices based on basic features.
